  # AI Education Analyzer Model

  A machine learning project that analyzes educational data from 10 countries
  to predict student employment outcomes and academic performance.

  ## What it does
  Compares three ML approaches on real-world education metrics — PISA scores,
  teacher quality, student stress levels, and employability indicators —
  to predict student outcomes and identify key influencing factors.

  ## Models Implemented
  - **Linear Regression** (OLS, Ridge, Lasso) — best: OLS with top 3 features (LOO-CV RMSE: 0.6372)
  - **Random Forest** — with hyperparameter tuning and permutation importance analysis
  - **XGBoost** — best performer (R² = 0.9951, MSE = 0.0168)

  ## Tech Stack
  - Python · Jupyter Notebooks
  - scikit-learn · XGBoost · pandas · NumPy
  - Matplotlib · Seaborn

  ## Key Concepts Demonstrated
  - Leave-One-Out Cross-Validation (LOO-CV) for small datasets
  - Feature importance (MDI vs Permutation-based)
  - Grid Search hyperparameter tuning
  - Model comparison and evaluation

  ## Who is this for?
  Students learning ML fundamentals, data science practitioners working with
  small datasets, and researchers interested in education analytics.
